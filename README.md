## Get started

#### Install Dependencies

- Open App-Development\App\App\ in your terminal
- Run these commands:

   ```bash
   npm install
   npm install expo@latest
   npx expo install --fix
   ```

#### Start The App

- Run this command:

   ```bash
   npx expo start --tunnel
   ```

#### Use The Expo Go App

- Download the Expo Go App
- Scan the QR code in the terminal
